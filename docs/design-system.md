# Design System - El Maestrito

## Colores (Tokens)
- **Backgrounds**: 0 (Negro), 1, 2, 3 (Grises técnicos)
- **Inks**: 0 (Blanco), 1, 2, 3 (Grises de texto)
- **Accent**: #FACC15 (Amarillo)
- **Status**: Danger (#EF4444), Success (#10B981)

## Tipografía
- **Primaria**: Inter (para UI general)
- **Mono**: JetBrains Mono (para datos técnicos y IDs)

## Componentes UI
- Logo
- Icon (Custom SVG set)
- Button
- Field
- Avatar
- Badge
- Stars
- Card
- Placeholder
- ScreenHeader
